# こうかとんが転んだ

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* こうかとんがこちらを見てくるタイミングで動かないようにして、無効を向いている間にこうかとんのところまでたどり着こう！

## ゲームの遊び方
* 矢印キーでプレイヤーを操作して、こうかとんのいるところまでたどりつけばクリア
* こうかとんがこちらを向いているときに動いてしまうと、ゲームオーバーとなる

## ゲームの実装
### 共通基本機能
* 左から右に向かう
* タッチしたらクリア
* 振り向いているときに動いたらゲームオーバー
* 不定期で振り向く
* リスタート機能
* 音声
* 足音

### 分担追加機能
* マップ上に遮蔽を設置 (担当:石川) 
	- 鬼が後ろを向いた瞬間、指定範囲内に1~3 個の遮蔽物をランダム生成
	- プレイヤーの移動時にX/Y方向ごとの衝突判定を実装し、めり込みを防止 
	- 1つ前の生成位置を記録し、遮蔽物同士が密集しないように配置を調整  
	- プレイヤーが遮蔽物に上下 35px 以上重なっている場合は"隠れている"と判定
* 時間停止&経過時間の表示
    - 60秒経過でゲームオーバー
* ライフ
* クリア後次のステージにいって難易度が上がる
* スタミナ
